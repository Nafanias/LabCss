Json в JavaScript

Примечание: в Python данные JSON обычно представлены в виде строки.

1. Парсинг JSON в JavaScript

Парсинг JSON — это процесс преобразования строки JSON в нативный JavaScript объект

. JavaScript предоставляет метод JSON.parse()
для этой цели.

const json = '{"name": "Ravi", "age": 25}';
const obj = JSON.parse(json);
console.log(obj.name);

Результат

Ravi

  • jsonData — это строка JSON.
  • Метод JSON.parse() преобразует её в объект JavaScript, и вы можете получить доступ к свойствам, таким как name и age.

Обработка исключений при парсинге JSON

const json = '{"name": "Ravi", "age": 25';

try {
    const obj = JSON.parse(json);
    console.log(obj.name);
} catch (error) {
    console.error("Error parsing JSON:", error);
}

В этом примере отсутствующая фигурная скобка вызывает ошибку. Блок catch помогает вам элегантно обработать такие проблемы.

2. Stringifying JavaScript объектов

После того как вы распарсили JSON и преобразовали его в объект JavaScript, вы можете захотеть отправить его обратно в виде строки JSON. Это делается с помощью JSON.stringify()

.

const user = {
    name: "Ravi",
    age: 25,
    profession: "Web Developer"
};

const jsonS = JSON.stringify(user);
console.log(jsonS);

Результат

{"name":"Ravi","age":25,"profession":"Web Developer"}

Это преобразует объект user в строку, которую можно отправить по сети или сохранить в файл.

3. Модификация данных JSON

После того как вы распарсили JSON строку

в JavaScript объект
, вы можете легко изменить его свойства или добавить новые.

const json = '{"name": "Ravi", "age": 25}';
const obj = JSON.parse(json);

obj.profession = "Web Developer";
console.log(JSON.stringify(obj));

Результат

{"name":"Ravi","age":25,"profession":"Web Developer"}

Здесь мы добавляем новое свойство (profession) к распарсенному объекту JSON и преобразуем его обратно в строку JSON с помощью JSON.stringify().

4. Фильтрация и преобразование данных JSON

Вам часто может понадобиться фильтровать или преобразовывать данные, содержащиеся в объектах или массивах JSON. Методы массивов JavaScript, такие как filter()

, map()
и reduce()
, идеально подходят для этой задачи.

Фильтрация данных JSON

Предположим, у вас есть массив объектов пользователей, и вы хотите отфильтровать пользователей, старше 30 лет.

const users = [
    { "name": "Amit", "age": 24 },
    { "name": "Rahul", "age": 45 },
    { "name": "Priya", "age": 30 }
];

const olderThan30 = users.filter(user => user.age > 30);
console.log(olderThan30);

В этом примере метод filter() используется для создания нового массива, который включает только пользователей старше 30 лет.

5. Сериализация объектов в формат JSON

Сериализация — это процесс преобразования объекта JavaScript в строку JSON. Это обычно делается при отправке данных с клиента на сервер.

const user = {
    name: "Simran",
    age: 29
};

const jsonS = JSON.stringify(user);
console.log(jsonS);

Результат

{"name":"Simran","age":29}

Это делает данные готовыми для передачи в формате JSON через HTTP-запросы или сохранения в системе хранения.

6. Настройка сериализации JSON

JSON.stringify()

позволяет вам указать, какие свойства включать или исключать из строки JSON, используя функцию замены.

const user = {
    name: "Jatin",
    age: 32,
    password: "123456"
};

const replacer = (key, value) => {
    if (key === "password") {
        return undefined; // Exclude password
    }
    return value;
};

const jsonS = JSON.stringify(user, replacer);
console.log(jsonS);

Результат

{"name":"Jatin","age":32}

В этом примере поле пароля исключается из окончательной строки JSON.

7. Использование JSON с API

JSON является стандартным форматом для обмена данными с веб-API. JavaScript fetch() API

в позволяет легко отправлять и получать данные JSON.

Получение JSON из API:

fetch('your-api-url')
    .then(response => response.json())
    .then(data => console.log(data))
    .catch(error => console.error('Error:', error));

Это получает данные из примера API и выводит их в консоль.

Отправка JSON в API:

const pData = {
    name: "Ravi",
    age: 25
};

fetch('your-api-url', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json'
    },
    body: JSON.stringify(pData)  // Send JSON data
})
    .then(response => response.json())
    .then(data => console.log('Success:', data))
    .catch(error => console.error('Error:', error));

В этом примере мы отправляем данные JSON в API с использованием метода POST.

Лучшие практики работы с JSON в JavaScript:

  • Обработка ошибок: Всегда используйте try...catch для обработки ошибок при парсинге JSON.
  • Используйте JSON Schema: Валидируйте структуру JSON с помощью JSON Schema, чтобы обеспечить целостность данных.
  • Минимизируйте данные: Избегайте ненужных данных в JSON. Используйте функцию замены JSON.stringify() для исключения конфиденциальной информации, такой как пароли.
  • Используйте библиотеки: Для манипуляции с JSON в большом масштабе библиотеки, такие как Lodash, могут помочь упростить задачи, такие как глубокое клонирование или слияние объектов JSON.
